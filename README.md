# BRAIVE Ads

Multi-channel ad campaign creation and client approval portal. Built for BRAIVE.

## Stack
- Next.js 14 (App Router)
- TypeScript
- React 18
- Plain CSS (no Tailwind yet)

## Setup
```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Status

**v0.1 (current)** — Brief + Architect stages. Dark instrument design (Direction A). Mock data only.
